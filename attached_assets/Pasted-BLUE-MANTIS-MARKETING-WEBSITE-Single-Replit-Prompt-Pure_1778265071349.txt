BLUE MANTIS — MARKETING WEBSITE
Single Replit Prompt · Pure Frontend · No Backend Required
==========================================================

WHEN TO USE THIS:
Paste the prompt below into Replit AI as a single message.
It builds the entire marketing website in one pass.
No Supabase. No backend. All form submissions use mailto: links.
The only output is a production-ready static React site.

==========================================================
THE PROMPT — COPY EVERYTHING BELOW THIS LINE
==========================================================

Build a complete, production-grade marketing website for Blue Mantis
(bluemantis.io) as a single-page React application using Vite + React +
TailwindCSS. No backend. No database. All CTA buttons either scroll to
a section or open mailto: links.

Install these dependencies:
  react, react-dom, vite, tailwindcss, @tailwindcss/vite
  react-router-dom (for the /security sub-page)
  react-intersection-observer (for scroll animations)

  Steps:
    1  Developer picks a task
       From JIRA or Azure DevOps. All incomplete epics, stories, tasks
       and bugs in one prioritised view.
    2  Blue Mantis reads the task
       Title, description, and acceptance criteria — automatically parsed.
    3  Finds relevant files
       Searches the connected Git repository using context matching.
    4  Four AI agents generate code simultaneously
       Claude, GPT-4o, Anti Gravity, and MS Copilot — all in parallel.
    5  Agents debate and critique each other
       Structured review. Scored on correctness, readability, minimal diff,
       and convention adherence.
    6  Developer reviews side-by-side diffs
       Four tabs. One recommended. Plain-English explanation of why.
    7  One click — done
       PR opened. Code committed. Ticket closed automatically.

  Auto-advance: useEffect with setInterval(2800ms) cycles through steps 0–6
  then loops. Pauses on hover (clearInterval on mouseEnter, resume on mouseLeave).
